# Tool Map

FFD MCP favors stable high-value workflows over a large number of scattered endpoints.

## First Call

If you are not sure which tool to use, call:

- `ffd_route_plan`
- `ffd_capabilities`

## Research Workflows

- Market intelligence: `ffd_market_intelligence`
- Money flow: `ffd_money_flow`
- Global news: `ffd_news_latest`, `ffd_news_search`
- Research reports: `ffd_research_search`, `ffd_research_detail`, `ffd_research_download`
- Funds: `ffd_fund_search`, `ffd_fund_profile`, `ffd_fund_market_performance`, `ffd_fund_nav_history`, `ffd_fund_portfolio`
- Industry and sector: `ffd_industry_stocks`, `ffd_industry_overview`, `ffd_industry_signal`
- Macro: `ffd_macro_data`
- Technical and risk: `ffd_technical_indicators`, `ffd_risk_metrics`, `ffd_backtest_ma_cross`

See [../skill.md](../skill.md) for the full agent-facing guide.
