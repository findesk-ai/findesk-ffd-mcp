# Trial And Billing

Current public trial for new registered users:

- 3-day trial
- 50,000 data points
- 5 research-report downloads
- 3-day global-news access

The MCP server does not grant anonymous credits locally. It uses your Findesk API Key, and the server checks the trial/subscription state on each protected request.

When a trial expires or quota is exhausted, FFD MCP should return a clear message with the next action: log in to Findesk, renew the subscription, or purchase a suitable plan.

Register: [https://www.findesk.cn/register](https://www.findesk.cn/register)
